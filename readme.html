How to use this page

  1. Set a number of dots and divide ratio with corresponding spin box
  2. Press enter key, and the dots will be displayed. You can click start button instead of pressing enter key
  3. Count the number of dots with pushing up key. Number of times of pushing up key is automatically counted
  4. Press enter key again immediately when you finished counting, then the result would be appeared in the text area for result
  5. Click download button above the textarea if you want to save the results
